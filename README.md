# Tauri + Scala.js
This is a direct porting of tauri with vanilla javascript demo to scala.js

### Scala.js compile
1. run sbt
1. use fastCopy / fullCopy command for compiling and copying of result js file to src directory

### Tauri
1. npm install
1. npm run tauri dev
