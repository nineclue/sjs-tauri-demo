# Tauri + Scala.js
This is a direct porting of tauri with vanilla javascript demo to scala.js

### Run
1. git clone https://github.com/nineclue/sjs-tauri-demo.git
1. cd sjs-tauri-demo
1. npm install
1. sbt fastCopy
1. npm run tauri dev
